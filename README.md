# TON Faucet Casino

Веб-приложение на Flask с раздельными страницами в одном меню:

- Кран TON (сбор **раз в 1 час**)
- Казино с **7 играми**
- Система буста дохода (чем больше общий объём ставок, тем выше награда крана)
- Реферальная система
- Пополнение и вывод через FaucetPay (демо-режим)
- Майнинг с окупаемостью **100 дней**

## Запуск локально

```bash
pip install -r requirements.txt
python app.py
```

Откройте: `http://127.0.0.1:5000`
